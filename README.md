# ecommerceProject


## Features
  
  - Login/Sign up User
  - User Profile
  - Admin Profile
  - Cart
  - Wishlist
  - Reviews
  - Orders
  - Checkout
  - Wallet
  - Search 
  - Categorization
  - Order Tracking
  - Admin Privileges
    - Add/remove categories
    - Blacklist Users
    - View/Deliver all Orders
    - Add/remove Coupons
    - Delete Products
    - Add Product
    - Edit Product

<br/>

## Technologies Used

- React
- NodeJS (Express)
- MS SQL
- Material UI
